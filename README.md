## Overview

- Interactive web application designed to inform users on the eyewear trends on every country where Essilor lenses are sold.

- Uses an interactive 3D globe similar to Google Earth that displays clickable pins per country to view data.

## Tech Stack

| Area               | Choice                                             |
| ------------------ | -------------------------------------------------- |
| Framework          | Next.js 15 (App Router)                            |
| Language           | TypeScript                                         |
| Styling            | SCSS Modules                                       |
| 3D / Visualization | `globe.gl` + Three.js                              |
| Animations         | CSS Animations, View Timelines, Web Animations API |
| Deployment         | Vercel                                             |
